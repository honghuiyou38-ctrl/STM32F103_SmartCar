# 基于STM32F103的智能车（STM32f103_smartcar_v1.0）

## MCU:STM32F103C8T6 (时钟树HCLK：72MHz)

## 项目简介
项目目前为初代版本，基于stm32f103开发，目前实现了简单的蓝牙遥控，电机驱动，OLED显示，以及小车循迹与避障功能，陀螺仪姿态解算，可作为智能小车的基础平台。

## 项目特点
• STM32 HAL库开发
• UART DMA + 空闲中断接收蓝牙数据
• 硬件I2C驱动OLED
• 硬件I2C驱动MPU6050
• TB6612双路PWM调速
• 模块化代码设计

---

## 功能
- 蓝牙遥控
- 硬件I2C驱动OLED显示
- TB6612电机驱动
- 按钮切换模式
- 八路循迹
- MPU5050陀螺仪转姿态解算
- HC-S04超声波避障

## 硬件：

### 一、电机驱动 (TB6612)

| 功能 | MCU 引脚 | 说明 |
|------|---------|------|
| **右电机 PWMA** | PA6 (TIM3 通道CH1) | PWM 信号，周期:10ms|
| **左电机 PWMB** | PA7 (TIM3 通道CH2) | PWM 信号 |
| **右电机方向 AIN1** | PB12 | GPIO 输出, 高=正转/低=反转 |
| **右电机方向 AIN2** | PB13 | GPIO 输出 |
| **左电机方向 BIN1** | PB14 | GPIO 输出 |
| **左电机方向 BIN2** | PB15 | GPIO 输出 |
> TB6612 逻辑: AIN1=H + AIN2=L → 正转, 反之反转。PWM 由 HAL 库直接输出。

---

### 二、八路循迹模块
| 传感器 | MCU 引脚 | 车身位置 |
|--------|---------|---------|
| **X1 (最左)** | PA1 | 左 1 |
| **X2** | PA2 | 左 2 |
| **X3** | PA3 | 左 3 |
| **X4** | PA4 | 左 4 |
| **X5** | PA5 | 右 4 |
| **X6** | PB0 | 右 3 |
| **X7** | PB1 | 右 2 |
| **X8 (最右)** | PA15 | 右 1 |
> 黑线=低电平(GPIO 读 0), 白底=高电平。安装在车头最前端, 距后轮约 20cm。

---

### 三、OLED显示屏 (SSD1306, 128×64, 硬件IIC)

| 功能 | MCU 引脚 | 说明 |
|------|---------|------|
| **SCL** | PB6 | 硬件I2C1时钟线 |
| **SDA** | PB7 | 硬件I2C数据线 |
| **VCC** | 3.3V | — |
| **GND** | GND | — |

---

### 四、蓝牙模块 (HC-05/06)

| 功能 | MCU 引脚 | 说明 |
|------|---------|------|
| **RXD** | PA9 | usart1:TX|
| **TXD** | PA10 | usart1:RX|
| **VCC** | 5V | — |
| **GND** | GND | — |

>通信方式：
- USART1（115200 bps）
- UART DMA + 空闲中断接收
- 支持 front、back、left、right、stop 以及MPUON、MPUOFF等控制指令

### 八、按钮与LED
| 功能 | MCU 引脚 | 触发方式 | 效果 |

| **按键KEY1 (切任务)** | PB5 | 低电平按下 | LED2亮起/熄灭&开启/关闭避障模式 |
| **LED1** | PA12 | 高电平亮 | 状态指示 |
| **LED2** | PA11 | 高电平亮 | 状态指示 |
