# 基于STM32F103的智能车（STM32f103_smartcar_v1.0）

## MCU:STM32F103C8T6 (时钟树HCLK：72MHz)

## 项目简介
项目目前为初代版本，基于stm32f103开发，目前实现了简单的蓝牙遥控，电机驱动，OLED显示，以及小车锁定功能，可作为智能小车的基础平台。

## 项目特点
• STM32 HAL库开发
• UART DMA + 空闲中断接收蓝牙数据
• 硬件I2C驱动OLED
• TB6612双路PWM调速
• 模块化代码设计

---

## 功能
-蓝牙遥控
-硬件I2C驱动OLED显示

## 硬件：

### 一、电机驱动 (TB6612)

| 功能 | MCU 引脚 | 说明 |
|------|---------|------|
| **右电机 PWMA** | PA6 (TIM3 通道CH1) | PWM 信号，周期:10ms|
| **左电机 PWMB** | PA7 (TIM3 通道CH2) | PWM 信号 |
| **右电机方向 AIN1** | PB12 | GPIO 输出, 高=正转/低=反转 |
| **右电机方向 AIN2** | PB13 | GPIO 输出 |
| **左电机方向 BIN1** | PB14 | GPIO 输出 |
| **左电机方向 BIN2** | PB15 | GPIO 输出 |
> TB6612 逻辑: AIN1=H + AIN2=L → 正转, 反之反转。PWM 由 HAL 库直接输出。

### 四、蓝牙模块 (HC-05/06)

| 功能 | MCU 引脚 | 说明 |
|------|---------|------|
| **RXD** | PA9 | usart1:TX|
| **TXD** | PA10 | usart1:RX|
| **VCC** | 5V | — |
| **GND** | GND | — |

>通信方式：
- USART1（115200 bps）
- UART DMA + 空闲中断接收
- 支持 front、back、left、right、stop 等控制指令
